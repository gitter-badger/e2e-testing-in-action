Selenium/Protractor in action

If you kick off some tests in selenium or protractor chances are that they run smoothly in one browser but not even start in the other. The reason for that can be anything and more often than not, hard to solve. That is why I started this project on GitHub to demonstrate how you can use selenium or protractor with different browser and run test locally or remote.

Prerequisites

Before we can do anything we need to download some browser drivers, create a browserstack account and of course download the demo project. In order to run the demo site too you need to have docker.
Lets first checkout the demo project 

$> git clone https://github.com/scaljeri/selenium-protractor-browserstack.git
